# 原力叙事 · 写作攻略（生长中）

> 这份攻略**每写一篇就长一点**——参照原力OS 绿皮书的「攻略 + 作业 + 能力架构」，把每次复盘的发现沉淀成可复用的纪律 + 能力清单。越往后，写作策略和能力越丰富。

## 脊柱：FTF（共鸣 → 可信 → 共识）
- **F 共鸣**：情感切入（反认知/反直觉/情感故事）→ 让读者「咦」+「这说的就是我」。入口。
- **T 可信**：数据 + 论文 + 案例 + **博古通今历史举证** → 让读者信不是忽悠。地基。
- **F 共识**：人文升华（东方智慧收口）→ 达成共识、愿转发。出口。
> 少一段就垮：缺 F1 没人进、缺 T 没人信、缺 F2 没人传。

## 评分逻辑（4 规矩）
1. 信号驱动非印象（每维 = 可数可查信号加权）。
2. 每信号挂出处（source_ref，诚实可追溯）。
3. 信号灯 🟢🟡🔴 诚实降级（低就标红 + surface）。
4. **自评 vs 人类打分的差距 = 进化信号**（差距最大维 = 盲区 → backlog → 下次动笔前先读）。
