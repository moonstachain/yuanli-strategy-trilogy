YUANLI NARRATIVE MODEL · 方法论详解

原力叙事模型

这不是一份「怎么写好文章」的通用指南。它只做一件事：把原力体系那套独特的「讲法」，从老师脑子里碰巧能讲对的状态，固化成一套谁拿起来都能照着复刻的生产纪律。所有例句都来自白皮书、通关地图、已发成稿——一字不改。
诚实的人＋冷硬的账＋东方的根＝焊成一篇
壹 三层混编声纹　·　贰 深入浅出5工艺　·　叁 娓娓道来5段节奏　·　肆 7母意象库　·　伍 人文底色　·　陆 疯传STEPPS　·　柒 四达标自检
〇一句话定义
先拿到整个模型的钥匙
原力叙事 ＝ 用同行者的诚实第一人称，把一条哲学公理拆成一张谁都能对标的硬账，再用东方智慧的底色收口——让读者既看穿了内幕、又算清了账、还被点醒了做人，然后忍不住转给下一个人。

它不是科普（科普只讲对错，不讲「为什么非这样不可」），不是鸡汤（鸡汤只给情绪，不给可对标的账），也不是纯硬核拆解（硬核只有冷数字，没有人味）。它是三样东西焊在一起：诚实的人 + 冷硬的账 + 东方的根。少任何一样，它就退化成那三种里的某一种——而那三种，市面上到处都是，不值钱。

★总脊柱 FTF · 共鸣 → 可信 → 共识
从读者体验看，整条叙事是一条三段路——下面所有工艺都是这三根支柱下的细则，评分也按这三维
F共鸣 · Feelings入口

情感切入反认知 / 反直觉 / 情感故事 → 让读者「咦，跟我想的不一样」+「这说的就是我」。＝钩子+张力段 · 工艺①问句②数字反差⑤故事弧 · STEPPS 情绪+故事 · 一层诚实(同行者)。

不共鸣，读者划走——后面再硬的账他也不看。
T可信 · Trust地基

权威支撑数据 / 论文 / 案例 ＋ 博古通今历史举证 → 让读者「这是真的、有据、不是你忽悠我」。＝框架+落地段 · 二层冷硬账 · canon 逐字 · 工艺③④ · ＋古今对照。

光共鸣是煽情——接上数据 + 历史举证，他才信不是忽悠。硬信号：每个核心论点挂 ≥1 古今对照·跨度≥50年·带出处
F共识 · Future出口

人文升华东方智慧收口 → 让读者「我懂了，也认了——该这么做人」。＝升华段 · 三层东方底色 · 回到起点升一维。

光可信是说教——升华到做人，他才认同并忍不住转发。

博古通今 ＝ T 可信支柱的历史举证层：今天这个理，历史上在哪个周期已经演过一遍——把"我说的"变成"历史早就证过的"，是降信任成本最狠的一招。共鸣是入口，可信是地基，共识是出口；少一段就垮。

壹三层混编声纹
这是原力叙事的指纹——别的都可仿，这个仿不像。独特不在任何单一一层，在「三层同时在场、互相咬合」
第一层第一人称诚实思考同行者，不是权威

定义作者站在读者旁边，不站在对面。讲的是「我也走过、我替你踩过坑」，不是「我懂你不懂、听我的」。最高级的标志是主动暴露自己的不确定——哪里是亲耳听的、哪里是我推的，划得清清楚楚。

为什么AI 时代权威贬值，读者对「专家口吻」天然设防。你越像走在前面半步的同路人，读者越卸防。诚实声明不是示弱，是最高级的建信任——一个肯说「这块我没实测」的人，他说「这块我实测了」时你才信。

说个我自己也想了很久才想通的事。有很长一段时间，我把「东西卖不上价」这件事，理解成了「手艺还不够好」。成稿开篇 —— 不是「我来教你」，是「我也困惑过」
这里我必须诚实声明一句……除了超级 VIP 那个量级是她亲口说的，其余我没有逐项拿到精确数字……我不拿推断冒充实测。成稿 —— 主动给方法论划边界
第二层数学/财务冷硬拆解把哲学落成可对标的账

定义所有抽象的「道」，必须能落成一张可以硬碰硬的账、一个可以算的数、一条恒真的定律。不停在形容词（「很厉害」「差很多」），一定要给到数量级（「100 倍」「恒真」「归零」）。

为什么这是原力叙事跟所有「人文写作」的分水岭。光有诚实和底色，会飘成情怀文；接上冷硬的账，读者就有了可以拿回去对标自己的尺子。情怀让人感动一次，账让人改一次行为。

同样一件东西，别人的边际成本在往上走，她的在往下掉。这一上一下，就是那一百倍差距真正的发动机。成稿 —— 把「为什么差100倍」落成一个可指的机械结构
复利压倒单利，进化压倒不变，飞轮压倒手推……这在数学上是恒真的，不依赖任何价值判断。白皮书 §1.3 —— 把公理钉成定律，不是观点
第三层东方智慧人文底色讲完账，回到做人

定义在冷硬的账之上，盖一层东方处世智慧——卖艺不卖身、善战者无赫赫战功、阴影即力量。它管的不是「怎么赚」，是「以什么姿势赚，才不把自己赔进去」。

为什么纯冷硬的账会让人变成精算的机器，读者跟你走一段就累。东方底色给整套体系留了一口气、一条命——它是原力叙事的「魂」，也是它区别于一切纯商业方法论的地方。

其实就一句话，尽量卖艺，不卖身。这句话我琢磨了很久。成稿 —— 把冷冰冰的风控点翻译成东方处世格言
阴影放到对的位置就成金色阴影（龟毛 → 挑剔产品质量）。白皮书概念14 —— 被压抑的负面，反面就是超能力

三层咬合的总检：一段好的原力叙事，往往是「诚实地引出一个问题（一层）→ 用一张账拆穿它（二层）→ 用一句东方智慧收口（三层）」。

贰深入浅出 · 5 工艺
让一公理沉到日常，让读者零门槛上车——不是天赋，是五个可复制的动作
①
问句开篇 + 一句话核心
用一个问句（或自白）把读者拽进来，紧接着甩出一句话就能记住的核心命题。先勾住，再交底。
「你说，我这东西到底值不值钱？」→ 引出整篇单位经济的拆解
什么时候用：每一节、每一篇的开头。一个问句定「核心问题」，一句话命题定「带走物」。
②
数字反差 1:100
找到那个最刺眼的数字反差，放到标题或开篇暴击。1:100、1000 倍、归零，比任何形容词都狠。
「一个超级客户一年顶一百个」「同样的能力，不同时代赚的钱差 1000 倍」
什么时候用：标题必用。找不到精确数字就找数量级，找不到数量级就找「归零」。
③
概念 → 隐喻 → 日常词
一个硬概念，先给隐喻，再给一个日常到不能再日常的词。三级一落，读者就懂了。
心理账户 → 口袋　·　ʸx → 药引（你睡觉时它还在长）　·　克制 → 卖艺不卖身
什么时候用：每出现一个体系术语，立刻问：它的隐喻是什么？日常词是什么？给不出日常词，读者记不住。
④
第一人称诚实声明
在最容易被质疑的地方（数据、推断、案例），主动停下来声明边界。「这块是她亲口说的，这块是我推的。」
「我不拿推断冒充实测。」
什么时候用：每当你给出一个看起来很硬、其实是推断的结论时。主动招认一次，买下读者对整篇的信任。
⑤
故事化推导 A→B→C
把一个道理拆成「问题 → 困境 → 转折 → 破局 → 结果」的弧线。不下结论让读者背，带读者走一遍，让他自己撞出结论。
最普通的门窗 → 卷不动 → 配出生证 → 换品类随身寺庙 → 抬进情绪/社交/投资账户
什么时候用：讲任何核心机制时。讲清「为什么非这样不可」，比喊一百句「你应该」管用。

陆疯传 STEPPS · 6 要素
现有成稿已把六要素全命中——但那是直觉碰巧打中的。模型的任务，是把每条变成写前主动埋、写后逐条核的纪律
要素它是什么原力叙事怎么落（主动埋的动作）
社交货币讲了显得你看穿了内幕给读者一个「内行才懂」的结构性洞察，让他想转给别人显摆
触发词日常场景能勾起你想到它把「边际成本/心理账户/30法则」做成读者生活里反复撞见的钩子
情绪理性愤怒 + 敬畏 + 同情混合「别再用战术上的勤奋，去掩盖战略上的偷懒」（理性愤怒）
公开性可被社交提及的公共标签用四关/12原型/58原子这种可命名、可标签化的体系做谈资
实用价值今天就能动手的东西永远给「今天能动手的三件事」+ 硬验收标准
故事每个故事都是「为什么错→怎么对」弧线每个案例都包装成「问题→困境→转折→破局」，不孤立讲数字
柒四达标自检
写完照着核，四达标缺一不发。任一维度不过，回去改
✅ 深入浅出（读者零门槛上车了吗）
  • 开篇有问句或自白当钩子？紧跟一句话核心命题？
  • 有至少一个 1:100 级的数字反差砸在标题或开篇？
  • 每个体系术语都给了隐喻 + 日常词？
  • 最容易被质疑的数据/推断处，有诚实边界声明？
  • 核心机制用「问题→困境→转折→破局」故事弧线讲，不是「你应该」命令句？
✅ 娓娓道来（节奏顺、能量曲线对吗）
  • 五段骨架齐全：钩子 → 张力 → 框架 → 落地 → 升华？
  • 张力段有反直觉判断（「真相恰恰相反」级别）？
  • 落地段给了「今天能动手的三件事」+ 硬验收标准？
  • 结尾回到开篇问句、升一维收口（首尾呼应）？
✅ 原力文脉（是不是这个体系的味道）
  • 全篇有一个主意象贯穿（7 选一，单峰纪律）？
  • 至少埋进 1 个 canon 母意象？
  • 引体系概念用了 canon 原话，没擅自改写？
✅ 人文底色（魂在不在）
  • 至少埋进 1 条东方底色（守拙/在场/松弛/觉醒/阴影），优先引 canon 原话？
  • 底色出现在「账」之后，收口升维，不是空洞前置的鸡汤？
  • 三层混编都在场：诚实第一人称 + 可对标的硬账 + 东方收口？
一票否决：三层混编缺任意一层（变成纯科普 / 纯鸡汤 / 纯硬核），或主意象超过一个（没记忆点），或东方底色前置成鸡汤——任一中招，回炉，不发。
原力叙事的终极标准，不是「读者读完觉得你很厉害」，
是「读者读完，看穿了一个内幕、算清了一笔账、还被点醒了怎么做人——然后忍不住转给下一个人」。
